# AAAzjzjzj Tech Notes

Personal technical documentation site built with [Docusaurus](https://docusaurus.io/). It collects notes, tutorials, and project write-ups around AI, Python, computer vision, and web development.

## Tech Stack

- Docusaurus 3
- React
- Markdown / MDX
- GitHub Pages

## Local Development

```bash
npm install
npm run start
```

## Build

```bash
npm run build
```

## Deployment

This site is configured for the custom domain:

```text
zjzj.click
```

Deploy with Docusaurus:

```bash
GIT_USER=AAAzjzjzj npm run deploy
```
