决策树

决策树是一种常用的分类和回归算法，它通过树形结构来表示决策规则。

算法原理​

决策树通过一系列的判断条件将数据集划分成不同的子集，每个内部节点表示一个特征上的测试，每个分支代表一个测试输出，每个叶节点代表一种类别。

决策树的构建​

特征选择​

决策树学习的关键是选择最优的特征进行划分。常用的特征选择标准有：

  1. 信息增益（ID3算法）
  2. 信息增益比（C4.5算法）
  3. 基尼指数（CART算法）

信息熵​

信息熵用于度量样本集合的纯度：

H(D) = -\sum_{k=1}^{K}p_k\log_2p_k

其中 p_k 是第 k 类样本所占的比例。

信息增益​

信息增益表示得知特征 A 的信息后使得类 Y 的信息不确定性减少的程度：

Gain(D, A) = H(D) - \sum_{v=1}^{V}\frac{|D^v|}{|D|}H(D^v)

代码实现​

from sklearn.tree import DecisionTreeClassifier
from sklearn.datasets import load_iris
from sklearn.model_selection import train_test_split
from sklearn.metrics import accuracy_score
import matplotlib.pyplot as plt
from sklearn import tree

# 加载数据集
iris = load_iris()
X = iris.data
y = iris.target

# 划分训练集和测试集
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.3, random_state=42)

# 创建决策树分类器
clf = DecisionTreeClassifier(max_depth=3, random_state=42)

# 训练模型
clf.fit(X_train, y_train)

# 预测
y_pred = clf.predict(X_test)

# 评估
accuracy = accuracy_score(y_test, y_pred)
print(f"准确率: {accuracy:.2f}")

# 可视化决策树
plt.figure(figsize=(12, 8))
tree.plot_tree(clf, feature_names=iris.feature_names, 
               class_names=iris.target_names, filled=True)
plt.show()

剪枝​

为了防止过拟合，需要对决策树进行剪枝：

预剪枝​

在构造过程中提前停止树的生长：

  • 限制树的最大深度
  • 限制叶节点的最小样本数
  • 限制节点划分的最小样本数

后剪枝​

先构造完整的决策树，然后自底向上地剪枝：

  • 代价复杂度剪枝
  • 错误率降低剪枝

优缺点​

优点​

  • 易于理解和解释
  • 可以处理数值型和类别型数据
  • 不需要数据预处理（归一化等）
  • 可以处理多输出问题

缺点​

  • 容易过拟合
  • 对数据的微小变化敏感
  • 可能产生偏向某些类别的树
  • 对于某些概念难以学习（如XOR问题）

应用场景​

  • 信用评分
  • 医疗诊断
  • 客户流失预测
  • 风险评估
